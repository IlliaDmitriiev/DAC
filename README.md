# DAC

How to build:
 - git clone https://github.com/IlliaDmitriiev/DAC.git
 - cd DAC
 - cmake .
 - cmake --build .
 
How to run:

 You should execute the following command after build in the same directory:
 ./DAC absolute_dir_path
 
Requires:
 - CMake 17 or latest
 - OS Linux
 - gcc compiler 7 or latest
